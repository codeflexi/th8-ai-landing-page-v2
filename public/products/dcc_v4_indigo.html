Pilot Implementation Notes

Prerequisites (IT)
  • Access to Invoice/PO tables (Read-only view)
  • Path to PDF Document Storage
  • User list for Role Mapping (RBAC)
Success Criteria
  • User สามารถตัดสินใจ (Approve/Hold) ได้ภายใน 8 ชม.
  • Audit สามารถ Export ประวัติการตัดสินใจได้ครบถ้วน
  • ลดเวลาในการค้นหาเอกสาร (Contract) ได้ > 50%
Out of Scope
  • No Write-back to SAP (เพื่อลดความเสี่ยง IT)
  • No AI/ML Prediction (ใช้ Rule-based 100% ในเฟสแรก)
